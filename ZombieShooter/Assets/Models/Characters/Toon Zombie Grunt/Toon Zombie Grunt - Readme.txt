Toon Zombie Grunt is a vertex colored character perfect for mobile as it is low poly, 
uses no textures and is rigged to work with Unity’s Mecanim system.

Asset Page - "http://chemicalbliss.com/wordpress/portfolio-items/toon-zombie-grunt/" 

Features

• Vertex colored Mecanim character
• 669 Tris
• Includes demo scenes and shaders
• Vertex Toon Shaders with many customization options
• 15+ Mecanim Animations
